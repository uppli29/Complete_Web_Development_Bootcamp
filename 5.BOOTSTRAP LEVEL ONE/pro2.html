Coffee Lover Project

Welcome to your Bootstrap Project:Coffee Lover

You will recreate this webpage and an additional Form.

Just follow the instruction steps here to re-create these webpages!

  1. Create a general landing page with a Jumbotron

  2. Next create a NavBar on the page that links another html file

  3. Add two paragraph of lorem-ipsum below the jumbotron

  4. Next use the Grid System to add in thumbnail pictures of coffee. Source:https://stocksnap.io/search/coffee. You will need to use the thumbail class for this. Here are the image links

    • One
    • Two
    • Three
    • Four
    • Five
    • Six

  5. Next make sure that largest settings should have 3 columns of images. On the smallest screen it should have 2 columns.It is upto you where you want to change

  6. Next create another html file for the signup page

  7. create form on the signup page with Email,password and checkbox

Thats it feel free to play around with this project with your own ides and images.The main focus of this project is to make use of navbar form grid
